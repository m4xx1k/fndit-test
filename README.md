## Як запустити вручну
### Фронтенд

```bash
# Перейдіть до папки frontend:
cd frontend

# Встановіть залежності:
yarn

# Запустіть розробку:
yarn dev
```

### Бекенд

```bash
# Перейдіть до папки backend:
cd backend

# Встановіть залежності:
yarn

# Запустіть розробку:
yarn dev

```
